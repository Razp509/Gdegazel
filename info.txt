Добавьте Docker Compose в автозапуск:
    Создайте файл crontab с помощью команды:
        bash
    crontab -e

В этом файле добавьте следующую строку для запуска Docker Compose при каждом запуске системы:
        bash
    @reboot docker-compose up -d

Это команда запускает Docker Compose в фоновом режиме (-d).

Проверьте задание Cron :
Чтобы убедиться, что задание cron настроено правильно, вы можете использовать команду для вывода списка текущих заданий cron:crontab -l
        bash
    crontab -l

Перезапустите службу Cron :
После добавления задания cron перезапустите службу cron, чтобы применить изменения:
        bash
    sudo service cron restart

Полностью прописанное задание с указанием местонахождения файла композ
@reboot /usr/local/bin/docker-compose -f /home/user/my-project/docker-compose.yml up -d
